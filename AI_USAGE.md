# AI usage disclosure

AI was used as a supporting production tool in this assignment.

| Area | How AI was used | Final human work |
| --- | --- | --- |
| Visual ideation | Explored a dual-domain art direction for humanoid robotics and rehabilitation technology. | Selected the direction, defined the visual hierarchy and integrated it into the page. |
| Image assets | Generated two original, non-branded campaign visuals: one humanoid robotics image and one rehabilitation exoskeleton image. | Chose the composition, placed, cropped and optimized the assets for the responsive design. |
| Code assistance | Assisted with drafting the initial Next.js, React, TypeScript and CSS implementation. | Reviewed, edited, structured and verified all code; built the responsive UI and made the final design decisions. |
| Copy refinement | Helped explore concise product and brand copy. | Curated and refined the final wording and content hierarchy. |

No external brand assets, copyrighted product photographs, proprietary code, or fabricated performance claims were used.
